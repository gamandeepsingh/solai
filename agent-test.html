Extension Info
How to use
  • 1Install SOLAI extension, unlock your wallet
  • 2Go to Agents tab → create an agent, set guardrails, fund it with SOL or USDC/USDT
  • 3Copy the Agent ID from the card and paste it in the forms below
  • 4Run any test — results appear inline and in the Activity Log
Agent ID (shared across all tests)
SOL Payment
window.solaiAgent.pay()
SPL Token Payment
window.solaiAgent.payToken()
Swap then Pay
window.solaiAgent.swapAndPay()

Agent swaps fromToken → toToken via Jupiter, then sends toToken to the recipient — all in one call. Requires mainnet.

Request Token Allowance
window.solaiAgent.requestAllowance()

Opens an approval popup in the extension. User grants a one-time spending limit so future payToken() calls within the limit don't need re-approval.

Guardrail Stress Tests
Expected to fail — verify guardrails are enforced

These calls should be rejected by the agent's guardrails. Useful to verify your guardrail config is working correctly.

Raw API — browser console
// SOL payment window.solaiAgent.pay({ agentId, recipient, amountSol, memo }) // SPL token payment (USDC, USDT, etc.) window.solaiAgent.payToken({ agentId, recipient, token, amount }) // Swap then send in one call window.solaiAgent.swapAndPay({ agentId, recipient, fromToken, toToken, toAmount }) // Request spending allowance (shows approval popup once) window.solaiAgent.requestAllowance({ agentId, token, maxAmount, expireDays, label })
