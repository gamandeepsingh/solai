## 📦 Install without building (Developer Mode)

The easiest way to try SOLAI — no Node.js or build tools needed.

**1. Download the extension**

- Go to the [Releases](../../releases) page on GitHub
- Download the latest `v1.0.4.zip`
- Unzip it anywhere on your computer — you'll get a folder called `dist/`

**2. Load into Chrome**

1. Open Chrome and go to `chrome://extensions`
2. Toggle **Developer mode** on (top-right corner)
3. Click **Load unpacked**
4. Select the unzipped `dist/` folder

SOLAI will appear in your extensions bar. Pin it for quick access.

> Works on Chrome, Brave, Arc, and any Chromium-based browser.

---

## ⚡ Getting Started (build from source)

```bash
pnpm install
pnpm build
```

Then load the built output:

1. Go to `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `extension/dist/` folder
