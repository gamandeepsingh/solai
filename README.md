## ✨ Overview

SOLAI Wallet is a **simple and friendly crypto wallet** focused on making Solana easy to use.  
Clean UI, playful design, and future-ready AI interactions.

---

## 🚀 Features (MVP)

- Solana wallet (create / import)
- View balance (SOL + SPL tokens)
- Send / Receive tokens
- Save contacts
- Minimal AI input UI (coming soon)
- Light + Dark mode

---

## 🔐 Security

- Non-custodial wallet
- Private keys stored locally
- No backend key access

---

## 🧠 Vision

Turn crypto into **simple commands**:

> "Send $5 to mom every day"  
> "Swap USDC to SOL"
> "Send 1SOL to Joe"

---

## ⚡ Getting Started

```bash
pnpm install
pnpm dev
```

Load extension in Chrome:

- Go to chrome://extensions
- Enable Developer Mode
- Click Load Unpacked
- Select dist/
